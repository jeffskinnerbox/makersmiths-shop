### How It Works
[Excalidraw][01] is an open-source, collaborative whiteboard tool designed to produce hand-drawn,
"sketch-style" diagrams and wireframes.
Excalidraw offers a[free whiteboard][02], can works offline, is end-to-end encrypted,
and lets you export your work as SVG or PNG easily.
It focuses on simplicity and speed, offering a minimalist interface that allows you to
quickly visualize ideas, flowcharts, or system architectures without the complexity of traditional design software.
Because it runs primarily in the browser, it is a popular choice for developers and teams who need a secure,
lightweight way to brainstorm or document concepts visually.

Working with Excalidraw offline is straightforward because it is built as a [Progressive Web App (PWA)][03],
meaning the website can actually "install" itself to your computer and run without an internet connection.
If you want a permanent offline instance that you control entirely, you can [run Excalidraw locally using Docker][04].
This is a great way to ensure you always have access to the tool regardless of their website status.
